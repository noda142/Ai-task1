# Image Recognition using Google Teachable Machine

## Objective
Train an image recognition model using Google Teachable Machine and classify images using Python.

## Classes
- Apple
- Banana

## Software
- Google Teachable Machine
- TensorFlow
- Python

## Steps

1. Create two image classes.
2. Upload training images.
3. Train the model.
4. Export the model as TensorFlow → Keras.
5. Download:
   - keras_model.h5
   - labels.txt
6. Place the files with predict.py.
7. Put the image as test.jpg.
8. Run:

python predict.py

## Output

The program prints:

Predicted Class:
Apple

Confidence:
0.98
